Docker如何安装

本项目建议使用 Docker 镜像！！！！！

# 镜像信息

  • 镜像名称：szzhoubanxian/reseed-puppy:latest

  • 镜像地址：https://hub.docker.com/r/szzhoubanxian/reseed-puppy

  • 前端端口：1919

# Docker CLI

docker run -d \
  --name=Reseed-Puppy-Latest \
  -v /数据库保存位置:/reseed-puppy-php/database \
  -v /你的qb种子目录:/reseed-puppy-php/public/qb \
  -v /你的qb1种子目录:/reseed-puppy-php/public/qb1 \
  -v /你的tr种子目录:/reseed-puppy-php/public/tr \
  -v /你的tr1种子目录:/reseed-puppy-php/public/tr1 \
  --network host \
  --restart unless-stopped \
  szzhoubanxian/reseed-puppy:latest

上面的种子目录映射首先需要找到对应种子文件夹然后映射到容器内部 上面举例了多个下载器如何映射

# Docker yaml

version: "3"
services:
  reseed-puppy:
    image: szzhoubanxian/reseed-puppy:latest
    container_name: Reseed-Puppy-Latest
    network_mode: host
    volumes:
      - "/数据库保存位置:/reseed-puppy-php/database"
      - "/你的qb种子目录:/reseed-puppy-php/public/qb" 
      - "/你的qb1种子目录:/reseed-puppy-php/public/qb1" 
      - "/你的tr种子目录:/reseed-puppy-php/public/tr" 
      - "/你的tr1种子目录:/reseed-puppy-php/public/tr1" 
    restart: unless-stopped

上面的种子目录映射首先需要找到对应种子文件夹然后映射到容器内部 上面举例了多个下载器如何映射

# 版本更新

  • 只要容器重启即可同步更新到最新版本，暂未开放自定义更新功能
