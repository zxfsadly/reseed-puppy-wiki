本项目建议使用 Docker 镜像！！！！

# Docker CLI

docker run -d \
  --name=Reseed-Puppy-Dev \
  -v /path/to/database:/reseed-puppy/database \
  -v /你的qb种子目录:/reseed-puppy/public/qb \
  -v /你的qb1种子目录:/reseed-puppy/public/qb1 \
  -v /你的tr种子目录:/reseed-puppy/public/tr \
  -v /你的tr1种子目录:/reseed-puppy/public/tr1 \
  --network host \
  --restart unless-stopped \
  szzhoubanxian/reseed-puppy:dev

上面的种子目录映射首先需要找到对应种子文件夹然后映射到容器内部 上面举例了多个下载器如何映射

# Docker yaml

version: "3"
services:
  reseed-puppy:
    image: szzhoubanxian/reseed-puppy:dev
    container_name: Reseed-Puppy-Dev
    network_mode: host
    volumes:
      - "/数据库保存位置:/reseed-puppy/database"
      - "/你的qb种子目录:/reseed-puppy/public/qb" 
      - "/你的qb1种子目录:/reseed-puppy/public/qb1" 
      - "/你的tr种子目录:/reseed-puppy/public/tr" 
      - "/你的tr1种子目录:/reseed-puppy/public/tr1" 
    restart: unless-stopped

上面的种子目录映射首先需要找到对应种子文件夹然后映射到容器内部 上面举例了多个下载器如何映射
