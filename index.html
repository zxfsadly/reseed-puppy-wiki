# 🎉 原理介绍

目前Reseed-Puppy支持辅种模式，【1】内置站点模式 【2】Jackeet辅种模式

# 1.内置站点模式

# 原理

基于NP版本1.8.5以上的pieces_hash接口判断种子在网站上是否存在，存在该种子则下载进行辅种。

# 优势

这是一个去中心化的辅种工具，无需等待工具适配站点，只需要站点是1.8.5版本以上以及未屏蔽接口即可进行辅种。

# 劣势

支持的站点目前只有43多个。

# 2.Jackett模式（稳定版目前支持）

# 原理

通过Jakett配置各个站点，然后通过检索站点资源进行相同种子辅种。

# 优势

无需站点适配，只要有站点账号即可进行辅种。

# 劣势

容易被站点流控、准确度也还不够高。
